# zenn-vscode-extension

VSCodeのWebエディタ（github.devなど）で、Zennの記事や本を執筆を補助するための[**VSCode Web 拡張機能**](https://code.visualstudio.com/api/extension-guides/web-extensions)です。

主な機能は以下の通りです。

- 記事や本をツリー表示
- 記事や本の雛形を作成
- 記事や本をプレビュー
